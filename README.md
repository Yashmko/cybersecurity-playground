# Cybersecurity Playground

Hands-on cybersecurity lab based on Cloud Security + Detection Engineering roadmap.

## Structure

01-foundations → networking, linux, python basics  
02-recon-enumeration → scanning and reconnaissance  
03-web-exploitation → DVWA and OWASP testing  
04-detection-engineering → logs, SIEM, MITRE mapping  
05-cloud-security → AWS security and misconfigurations  
06-automation-tools → custom scripts  
07-vulnerability-writeups → structured reports  
08-bug-bounty → real findings  

Goal:
- Build real-world cybersecurity skills
- Focus on detection engineering + cloud security
- Create portfolio-grade projects
